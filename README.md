# My Api GPT

## Installation

Clone the repository and install the dependencies.

```bash
git clone
cd nodejs-boilerplate
npm install
```

```bash
project
│ app.js
│ .env
│ package.json
│
└───controllers
│ exampleController.js
│
└───models
│ exampleModel.js
│
└───routes
│ exampleRoutes.js
│
└───middlewares
│ exampleMiddleware.js
│
└───helpers
exampleHelper.js
```
